# Quick Task 260710-9lc: Tắt deploy GitHub Pages

## Vấn đề

Repo đang deploy song song 2 nơi:
- Cloudflare → `valueinvesting.com.vn` (host chính, base `/`, canonical đúng)
- GitHub Pages → `vilo0411.github.io/investing` (bản trùng, canonical tự trỏ về github.io, `index,follow`)

Bản github.io là một bản sao đầy đủ, có index → gây duplicate content trên domain authority cao (github.io), cạnh tranh index / chia crawl budget với domain chính. Đây là nguyên nhân kỹ thuật cản trở việc auto-index của `valueinvesting.com.vn`.

## Task

1. Xoá `.github/workflows/deploy.yml` → chặn mọi lần GitHub Actions deploy lên Pages về sau.
2. (Thủ công — cần user) Tắt GitHub Pages trong repo Settings → Pages (Source = None) để bản đã publish trả 404. `gh` CLI không có sẵn trong môi trường nên bước này không tự động hoá được.

## Done

- `.github/workflows/deploy.yml` không còn tồn tại; không còn workflow nào deploy Pages.
- Cloudflare (`valueinvesting.com.vn`) không bị ảnh hưởng — nó build độc lập, không dùng workflow này.
