## Spacing Scale

Declared values (Phase 1 token system, `src/styles/tokens/spacing.css` — multiples of 4px, using rem):

| Token | Value | Usage |
|-------|-------|-------|
| `--space-1` | 4px (0.25rem) | Tight inline gaps (label-to-value, icon spacing) |
| `--space-2` | 8px (0.5rem) | Compact element spacing (list item gaps, expertise tag gaps) |
| `--space-3` | 12px (0.75rem) | Table cell padding (vertical) |
| `--space-4` | 16px (1rem) | Default element spacing, table cell padding (horizontal), avatar gap |
| `--space-6` | 24px (1.5rem) | Box padding (all callout components: Key Takeaways, Citation box, Disclaimer, AuthorBox) |
| `--space-8` | 32px (2rem) | (available, not required by this phase) |
| `--space-12` | 48px (3rem) | Top margin separating AuthorBox/callouts from preceding content |
| `--space-16` | 64px (4rem) | (available, not required by this phase) |
| `--space-24` | 96px (6rem) | (available, not required by this phase) |

Exceptions: none. All 6 new components (KeyTakeaways, CitationBox, AuthorBox v2, Breadcrumb, Disclaimer, ComparisonTable) use only the tokens above. Breadcrumb uses `--space-2` between items; ComparisonTable cells use `--space-3 --space-4` (12px/16px) per Pattern 3 in RESEARCH.md.

## Typography

All values from Phase 1 (`src/styles/tokens/typography.css`). This phase introduces no new font sizes/weights — components reuse the existing AuthorBox/body scale.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body / item text | `0.9rem`–`0.96rem` (~14.4–15.4px, matches `.author-bio`/`.author-note`) | 400 (regular) | 1.6–1.65 |
| Label (eyebrow/uppercase) | `0.78rem` (~12.5px) | 700 (bold) | 1.2 |
| Heading (component title — N/A, components use eyebrow labels not headings) | — | — | — |
| Table text | `0.92rem` (~14.7px) | 400 body / 700 header (`th`) | 1.5 |

**Declared scale for this phase (2 weights only, as required):**
- 400 (regular) — body text, list items, table cells, breadcrumb links, disclaimer text
- 700 (bold) — eyebrow labels (`.kt-label`, `.citation-label`, author labels), table header (`th`), comparison-table header text

Font sizes used (3, within 3–4 range): `0.78rem` (labels), `0.9rem`–`0.96rem` (body/list content), `0.92rem` (table). All inherit `var(--font-sans)` except author name (`var(--font-serif)`, pre-existing, unchanged).

Heading line-height: not applicable — none of the 6 components introduce a true `<h1>`–`<h3>`; "title" text uses the label/eyebrow style above (line-height 1.2).

Body line-height: 1.6 default for list items and paragraph text (consistent with `.author-bio`/`.author-note` in existing AuthorBox); table rows use 1.5 (denser, data-grid context per D-12).

## Component-Specific Visual Contracts

### 1. KeyTakeaways (`src/components/KeyTakeaways.astro`)
- Props: `items: string[]`
- `items.length === 0` → render `null` (no placeholder, no reserved DOM node)
- `items.length > 0` → `<aside class="key-takeaways" aria-label="Tóm tắt nội dung chính">`
- Visual: border-left 4px `var(--color-brand-900)`, `background: var(--surface)`, `border: 1px solid var(--line)`, `border-radius: var(--radius-md)`, `padding: var(--space-6)`
- Label: "Tóm tắt nội dung chính" — `font-size: 0.78rem; font-weight: 700; text-transform: uppercase; letter-spacing: 0.08em; color: var(--color-accent)`
- List: `<ul>` with `list-style: none`, `display: grid`, `gap: var(--space-2)`; each `<li>` prefixed with "✓ " then item text

### 2. CitationBox (`src/components/CitationBox.astro`)
- Props: `sources?: string[]`, `citations?: { title: string; url?: string; publisher?: string; date?: string }[]`, `factCheckedDate?: Date`, `updatedDate: Date`
- Render condition: `sources.length > 0 || citations.length > 0` → render `<aside>`; else `null`
- Visual: same box pattern as KeyTakeaways (border-left + surface + line + radius-md + space-6 padding)
- Date label rendered via `.toLocaleDateString("vi-VN")` (Pitfall 2 — never raw `Date` object)
- If `citations.length > 0` → render full entries (title, optional link with `target="_blank" rel="noopener noreferrer"`, optional ` — {publisher}`, optional ` ({date})`)
- Else if `sources.length > 0` → render each string as plain `<li>` (no link/publisher/date)
- Section heading above list: "Nguồn tham khảo"

### 3. AuthorBox v2 (`src/components/AuthorBox.astro`, modified in place)
- Adds: `author.credentials: string[]` (conditional block, D-08), `author.experience: string` (always shown when non-empty)
- Profile link target changes from `/about/` to `/author/{author.slug}` (both occurrences: name link and "Xem hồ sơ" button)
- Preserve all v1 elements unchanged: avatar initial circle, name, role, bio, expertise tags, author-note
- Credentials block (when present): sub-label "Chứng chỉ" (same eyebrow style as `.author-label` but smaller weight context — use `font-size: 0.78rem; font-weight: 700; color: var(--muted)` to differentiate from primary label), `<ul>` of credential strings
- Experience text: rendered as `<p class="author-experience">` styled like `.author-bio` (0.9rem, `var(--muted)`, line-height 1.6)

### 4. Breadcrumb (`src/components/Breadcrumb.astro`)
- Props: `items: { label: string; href?: string }[]`
- Renders `<nav class="breadcrumb" aria-label="Breadcrumb">` with `<ol>` of items
- Each item except last: `<li><a href={item.href}>{item.label}</a><span class="separator">›</span></li>`
- Last item: plain text (current page), `aria-current="page"`, no link even if `href` provided
- Visual: inline flex, `gap: var(--space-2)`, `font-size: 0.85rem`, links colored `var(--brand)` with `text-decoration: none` + hover underline; separator colored `var(--muted)`
- No box/border wrapper — breadcrumb sits flush, unlike the 4 callout components (it's a navigation element, not a callout)
- Must work identically for article context (`["Trang chủ", "Đầu tư", "Cổ phiếu", "{article title}"]`) and category context (`["Trang chủ", "Đầu tư", "{category name}"]`)

### 5. Disclaimer (`src/components/Disclaimer.astro`)
- Props: `text?: string` (defaults to `site.disclosure` from `src/data/site.ts`)
- Visual: same box pattern (border-left 4px `var(--color-brand-900)`, `background: var(--surface)`, `border: 1px solid var(--line)`, `border-radius: var(--radius-md)`, `padding: var(--space-6)`, `display: grid; gap: var(--space-1)`)
- `<strong>` "Lưu ý quan trọng:" colored `var(--color-brand-900)`
- `<span>` body text colored `var(--muted)`, `font-size: 0.96rem`, `line-height: 1.65`
- `aria-label="Lưu ý tài chính"` on the `<aside>`

### 6. ComparisonTable (`src/components/ComparisonTable.astro`)
- Props: `columns: string[]`, `rows: (string | number)[][]`, `caption?: string`
- Wrapper: `<div class="comparison-table-wrap" style="overflow-x: auto">` (D-11) — `border: 1px solid var(--line); border-radius: var(--radius-md); margin: var(--space-6) 0`
- `<table>`: `width: 100%; border-collapse: collapse; font-size: 0.92rem`
- `<thead><th>`: `background: var(--color-brand-900); color: #fff; text-align: left; padding: var(--space-3) var(--space-4); white-space: nowrap` (per D-12)
- `<tbody><tr:nth-child(even)>`: `background: var(--surface-alt)` (zebra, per D-12)
- `<td>`: `padding: var(--space-3) var(--space-4); border-bottom: 1px solid var(--line)`
- Optional `<caption>` rendered when `caption` prop provided
