- **Word Count Gap:** Bài hiện tại ~X từ — Đối thủ trung bình ~X từ → [Cần thêm ~X từ / Cần cắt ~X từ / Đang trong range].

### Đề xuất cập nhật SEO Elements
> Nếu bài gốc cần chỉnh Title hoặc Meta Description — ghi rõ tại đây TRƯỚC khi vào heading analysis. Nếu giữ nguyên — ghi "Giữ nguyên".

- **Title hiện tại:** [Tiêu đề hiện tại — X ký tự]
- **Title đề xuất:** [Tiêu đề mới — tối đa 59 ký tự, chứa target keyword] _(hoặc: Giữ nguyên)_
- **Meta Description hiện tại:** [Nội dung hiện tại — X ký tự]
- **Meta Description đề xuất:** [Nội dung mới — ~155 ký tự, chứa target keyword, có benefit/CTA] _(hoặc: Giữ nguyên)_
